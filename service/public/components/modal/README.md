# modal组件

## 组件描述

##组件应用
1. 引用蒙层(model)组件库
2. 被使用的元素加上class = zou-button

##应用举例
```html
普通按钮
<a href="#" class="zou-button">normal</a>
警告按钮
<a href="#" class="zou-button zou-button-warn">warn</a>
取消按钮
<a href="#" class="zou-button zou-button-cancel">cancel</a>
```

##所有类名
.zou-button           一般样式
.zou-button-warn      警告样式
.zou-button-cancel    取消样式
